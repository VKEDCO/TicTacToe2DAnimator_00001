1) Eclipse Project (TicTacToe2DAnimator_00001.rar) with an 
Android application that shows how to do 2D frame animations,
i.e., image sequences. Animations can be selected via
an options menu. The 1st animation shows a case of X winning. 
The 2nd animation shows a case of O winning. The 3rd animation 
shows a case of a draw.

2) Targets: Android 2.3+, Eclipse Helios 3.6.

3) To start: 
   a) unzip into a folder 
   b) open in Eclipse 
   c) Run as Android application
